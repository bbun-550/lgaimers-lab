# Notebooks: 생각의 실험실

이 디렉토리는 **"코드를 만들기 전, 생각을 검증하는 실험실"**입니다.
프로덕션 코드가 아닌, 자유로운 실험과 분석을 위한 공간입니다.

## 목적

Notebook은 **'결론을 코드로 옮기기 전 단계'**를 담당합니다.

1.  **가설 세우기 (Hypothesis)**: "이 레이어를 빼면 어떻게 될까?"
2.  **작은 단위 검증 (Validation)**: 전체 학습 대신 일부 데이터로 빠르게 확인
3.  **시각적 이해 (Visualization)**: 데이터 분포, 어텐션 맵 시각화
4.  **결과 해석 (Interpretation)**: 실험 결과의 의미 분석

## 주의사항
- 이곳의 코드는 정리되지 않아도 괜찮습니다.
- 하지만 **유의미한 결론**이 나오면 반드시 `src/` 디렉토리의 모듈로 **이관(Refactoring)**해야 합니다.
